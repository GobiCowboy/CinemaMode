## 记录规则

只记录可能被多个功能复用的实现方式，例如：
- 日志记录
- 文件解析
- 文件上传
- 导出生成
- 分页查询
- 权限判断
- 错误处理
- 配置读取
- 任务状态管理

不要记录每个普通函数。

## 状态选项

| 状态 | 含义 |
|------|------|
| 是 | 已抽象为公共模块，可直接复用 |
| 否 | 尚未抽象，仍分散在具体功能中 |
| 已废弃 | 不再使用 |
