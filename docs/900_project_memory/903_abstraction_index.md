# 903 待抽象索引

## 状态选项

| 状态 | 含义 |
|------|------|
| 观察中 | 首次出现，暂不处理 |
| 待抽象 | 已重复，等待抽象 |
| 已抽象 | 已完成抽象，更新至 902_implementation_index.md |
| 不抽象 | 决定不抽象，下方说明原因 |
| 已废弃 | 不再需要 |

---

| 逻辑名称 | 出现功能 | 重复次数 | 当前状态 | 建议处理 | 关联实现 |
|----------|----------|:--------:|----------|----------|----------|
| | | 1 | 观察中 | 暂不抽象 | |

## 记录规则

- **第一次出现**：可以不记录，除非明显会复用。
- **第二次出现**：必须记录到本文件。
- **第三次出现**：必须评估是否抽象。

### 抽象处理方式

1. 抽象成公共函数。
2. 抽象成公共组件。
3. 抽象成公共服务。
4. 暂不抽象——必须在表格「建议处理」列写明原因。

### 已抽象后

1. 更新 `902_implementation_index.md`。
2. 把本文件状态改为「已抽象」。
3. 更新相关功能文档的相关文件路径。
